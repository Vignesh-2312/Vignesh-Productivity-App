# Vignesh's Productivity App

A personal task manager built using Python, Flask, HTML/CSS, and SQLite. Categorise and manage your day with a clean, responsive interface.

## Features
- Add tasks with categories
- View tasks in organised cards (Groceries, Work, etc.)
- Toggle task completion 
- Delete tasks 
- Responsive design 

## Tech Stack
- Python
- Flask
- SQLite
- HTML, CSS

## How to Run Locally

1. Clone the repo  
   `git clone https://github.com/Vignesh-2312/Vignesh-Productivity-App`

2. Create and activate a virtual environment  
   `python -m venv venv`  
   `.\venv\Scripts\activate` (on Windows)

3. Install dependencies  
   `pip install -r requirements.txt`

4. Run the app  
   `python main.py`

5. Open in browser:  
   `http://127.0.0.1:5000`
